Política de Privacidade

Relógio do Apocalipse Global

Transparência e Proteção de Dados

O Relógio do Apocalipse Global, desenvolvido pela The Kavlac Group, é uma ferramenta de monitoramento estratégico que prioriza sua privacidade. Esta política explica como tratamos os dados no aplicativo.

Última atualização: Fevereiro de 2024

1. Dados Coletados

1.1 Dados Públicos de Fontes Externas

O aplicativo agrega dados públicos de agências científicas e governamentais:

  • NASA: Dados sobre asteroides e atividade solar
  • ESA: Informações espaciais e monitoramento climático
  • NOAA: Dados meteorológicos e climáticos
  • USGS: Atividade sísmica e vulcânica
  • ONU: Indicadores de crises governamentais e conflitos

Importante: Estes são dados públicos disponíveis para todos. O app apenas os agrega e processa.

1.2 Dados Técnicos do Dispositivo

Para garantir o funcionamento adequado do app, coletamos:

  • Modelo do dispositivo e versão do Android
  • Resolução de tela (para otimizar a interface)
  • Relatórios de crashes (anônimos)
  • Dados de performance da aplicação

Estes dados são completamente anônimos e usados apenas para melhorias técnicas.

1.3 Dados NÃO Coletados

O app NÃO coleta:

  • Localização GPS
  • Contatos ou lista telefônica
  • Mensagens ou comunicações
  • Fotos, vídeos ou arquivos pessoais
  • Dados de navegação em outros apps
  • Informações de identificação pessoal

2. Como Utilizamos os Dados

Os dados são utilizados exclusivamente para:

  • Exibir o relógio de contagem regressiva até 22/12/2032
  • Calcular e exibir o Índice de Risco Global
  • Mostrar os 8 vetores de ameaça atualizados
  • Otimizar a interface para seu dispositivo
  • Corrigir bugs e melhorar a estabilidade

Nós NUNCA:

  • Vendemos dados a terceiros
  • Compartilhamos informações com anunciantes
  • Rastreamos sua atividade fora do app
  • Coletamos dados pessoais identificáveis

3. Segurança e Arquitetura

Proteções Implementadas:

  • Arquitetura MVVM: Separação de lógica e interface para maior segurança
  • View Binding: Prevenção de falhas e acessos não autorizados à interface
  • Coroutines: Processamento seguro em segundo plano
  • Modo Escuro Nativo: Redução de consumo de bateria e melhor segurança visual

Armazenamento Local:

Os dados técnicos são armazenados localmente no seu dispositivo usando o sistema de sandbox do Android, garantindo isolamento de outros aplicativos.

4. Permissões do Aplicativo

O app solicita apenas as seguintes permissões:

  • Internet: Para buscar dados atualizados das agências (NASA, ESA, etc.)
  • Acesso à Rede: Verificar disponibilidade de conexão

Nenhuma outra permissão é solicitada. O app não precisa de acesso a câmera, microfone, contatos, localização ou arquivos.

5. Exclusão de Dados

Como Solicitar Exclusão:

  1. Envie email para: thekavlacgroup@gmail.com
  2. Assunto: "Excluir meus dados - Relógio do Apocalipse Global"
  3. Informe o modelo do seu dispositivo
  4. Prazo de processamento: até 7 dias úteis

O que será deletado:

  • Relatórios de crashes associados ao seu dispositivo
  • Dados técnicos anônimos (se identificáveis)

Nota: Como o app não coleta dados pessoais ou de conta, a exclusão é principalmente preventiva. Os dados públicos de agências (NASA, ESA, etc.) não são deletados pois não pertencem ao app.

6. Menores de Idade

O aplicativo é adequado para todas as idades e não coleta intencionalmente dados de menores de 13 anos. Como não coletamos dados pessoais, não há risco de identificação de menores.

7. Alterações nesta Política

Podemos atualizar esta política ocasionalmente. Alterações significativas serão notificadas através de uma atualização do app na Play Store.

A data de "Última atualização" no topo desta página sempre refletirá a versão mais recente.

8. Propriedade Intelectual

O Relógio do Apocalipse Global é propriedade intelectual da The Kavlac Group. A metodologia de cálculo do Índice de Risco Global e a arquitetura do sistema são protegidas por direitos autorais.
