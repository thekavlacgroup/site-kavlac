Perguntas Frequentes

Como funciona o sistema de categorias?

NOVO! O app agora possui categorias pré-definidas (Geral, Spam, Bancário, Delivery, Pessoal, Trabalho) e você pode criar categorias personalizadas ilimitadas. Pressione e segure uma mensagem, selecione "Categorizar" e escolha ou crie uma categoria.

Como filtrar mensagens por categoria?

NOVO! Use os chips coloridos no topo da tela para filtrar mensagens por categoria. Toque em uma categoria para ver apenas mensagens daquela categoria. A busca funciona dentro da categoria ativa.

Como fazer backup das minhas mensagens e configurações?

NOVO! Vá no menu (⋮) > Exportar Backup. O app criará um arquivo JSON com todas as suas categorias, números bloqueados e configurações. Você pode importar este backup em outro dispositivo ou após reinstalar o app.

Como restaurar um backup?

NOVO! Vá no menu (⋮) > Importar Backup e selecione o arquivo JSON que você exportou anteriormente. Todas as suas configurações, categorias e listas de bloqueio serão restauradas.

Como selecionar várias mensagens de uma vez?

NOVO! Pressione e segure uma mensagem para ativar o modo de seleção. Checkboxes aparecerão ao lado de cada mensagem. Toque nas mensagens que deseja selecionar e use as opções no menu para excluir ou categorizar em lote.

Como configurar dual SIM no app?

O app detecta automaticamente ambos os chips SIM. Use as abas "Chip 1" e "Chip 2" no topo para alternar entre as mensagens de cada operadora. O nome da operadora aparece na aba.

Como criar uma nova categoria personalizada?

NOVO! Pressione e segure uma mensagem > Categorizar > "+ Nova Categoria". Digite o nome e confirme. A categoria será criada e adicionada aos chips de filtro automaticamente.

Como excluir mensagens de uma categoria específica?

NOVO! Filtre pela categoria desejada usando os chips. Ative o modo de seleção (pressione e segure), marque as mensagens e use a opção de excluir. Isso afeta apenas mensagens da categoria ativa.

Como bloquear spam automaticamente?

O app possui filtro automático de spam com palavras-chave como "ganhou", "prêmio", "sorteio", "clique", "urgente", "oferta". Mensagens detectadas aparecem marcadas como [SPAM]. Você também pode marcar remetentes específicos como spam.

Como bloquear um número permanentemente?

Pressione e segure a mensagem do remetente > Bloquear Número. Mensagens deste número não aparecerão mais no app. Você pode desbloquear depois nas configurações se necessário.

Como usar a busca dentro de categorias?

NOVO! Primeiro, selecione a categoria desejada usando os chips. Depois, use a barra de busca no topo. A busca encontrará mensagens apenas dentro da categoria ativa, tornando os resultados mais precisos.

O que acontece após o período de 14 dias grátis?

Você terá 14 dias de acesso completo a todos os recursos. Após isso, pode optar por assinar o plano premium ou continuar usando recursos básicos. Suas configurações e categorias são mantidas.

Como atualizar a lista de mensagens?

Puxe a tela para baixo (pull to refresh) para recarregar todas as mensagens. O app sincronizará com o banco de dados do sistema e aplicará suas categorias e filtros.

Exclusão de Dados

Para solicitar a exclusão de sua conta e dados do SMS Dual:

  1. Use nosso formulário de contato
  2. Selecione "Privacidade e Dados" como assunto
  3. Escreva "Excluir meus dados - SMS Dual" na mensagem
  4. Informe seu email cadastrado no app (se houver)
  5. Aguarde confirmação (prazo: até 7 dias úteis)

Todos os backups (se houver) e dados sincronizados serão permanentemente removidos.

🎯 Dicas para Melhor Uso das Novas Funcionalidades

  • Organize desde o início: Categorize mensagens importantes assim que chegarem para manter tudo organizado
  • Crie categorias específicas: Além das padrão, crie categorias como "Família", "Compras", "Saúde" conforme sua necessidade
  • Use filtros + busca: Filtre por categoria primeiro, depois use a busca para encontrar mensagens específicas rapidamente
  • Faça backup regularmente: Exporte um backup a cada mês ou antes de trocar de dispositivo
  • Seleção múltipla para limpeza: Use o modo de seleção para limpar várias mensagens antigas de uma vez
  • Configure notificações diferentes para cada chip SIM: Identifique facilmente qual linha está recebendo mensagens
  • Ative o modo escuro à noite: Reduz cansaço visual e economiza bateria
  • Marque spam proativamente: Quanto mais você marcar, melhor o filtro automático ficará
  • Exporte antes de reinstalar: Se for resetar o celular, faça backup primeiro para não perder categorias
